Fetch Mapillary API data with your own API key and place it in this directory. The Jupyter notebook for fetching is `notebooks/fetch_mapillary_features.ipynb`.

Expected output files (one per country):

- `map_features_thailand.json`
- `map_features_maharashtra.json`

Set your token as `MAPILLARY_TOKEN` in a `.env` file at the repository root before running the notebook or `python src/fetch_mapillary_features.py`.
